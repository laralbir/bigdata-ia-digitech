# Python Essentials 1 — Cisco Networking Academy / OpenEDG Python Institute

## Resumen del curso

Python es un lenguaje utilizado tanto por nuevas empresas como por gigantes tecnológicos (Google, Facebook, Netflix, entre otros) para crear proyectos de software de cualquier escala. Gracias a su sintaxis sencilla y su código fácil de leer, es un excelente primer lenguaje de programación y no requiere conocimientos previos. Las habilidades adquiridas abren la puerta a carreras en prácticamente cualquier industria y son la base para roles más avanzados de ingeniería y desarrollo de software, como ingeniero de software, administrador de sistemas o ingeniero de seguridad.

El curso enseña a diseñar, desarrollar y depurar programas, a analizar problemas mediante programación y a aplicar buenas prácticas de codificación. Además, prepara para la certificación **PCEP — Certified Entry-Level Python Programmer** (examen PCEP-30-0x).
