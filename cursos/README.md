# Cursos

Este directorio recoge los **cursos adicionales** realizados antes o durante el curso lectivo del Curso de Especialización en Big Data e Inteligencia Artificial para FP (Digitech, 2026/2027): certificaciones, MOOCs y cursos de terceros que complementan el temario oficial de las [asignaturas](../asignaturas/). Cada curso tiene su propio subdirectorio, con la documentación generada a partir del material original siguiendo la convención descrita en el [CLAUDE.md](../CLAUDE.md) del repositorio.

## Índice de cursos

### [Fundamentos de Linux](fundamentos-linux/00-indice.md)

Introducción completa al sistema operativo Linux, tanto para uso de escritorio como para administración de sistemas: origen del kernel y distribuciones, línea de comandos y shell Bash, gestión de archivos, búsqueda y filtrado de texto, y fundamentos de scripting. La segunda mitad cubre administración de sistemas — hardware, paquetes, procesos, redes, y gestión de usuarios, grupos, propiedad y permisos (incluyendo permisos especiales y el estándar FHS). El temario cubre los objetivos de la certificación **LPI Linux Essentials** y sienta las bases para **LPIC-1** o **CompTIA Linux+**.

### [Fundamentos de Python 1](fundamentos_python_1/README.md) *(próximamente)*

Curso Python Essentials 1 de Cisco Networking Academy / OpenEDG Python Institute: introducción a Python y a la programación, tipos de datos y operadores básicos, valores booleanos, condicionales, bucles y listas, y funciones, tuplas, diccionarios y excepciones. Prepara para la certificación **PCEP**. Documentación de módulos pendiente de redacción.
